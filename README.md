
# sym-protein-val

## Installation

1. Install Boltz:
   ```bash
   cd modules/boltz
   pip install -e .[cuda]
   ```

2. Install FoldSeek:
   ```bash
   conda install -c conda-forge -c bioconda foldseek
   ```

3. Install remaining dependencies:
   ```bash
   pip install -r requirements.txt
   ```
